# Notes Management System

## Overview

The Notes Management System is a web-based application developed using HTML, CSS, and JavaScript. It allows users to create, edit, delete, search, and organize notes efficiently. The application stores data using the browser's Local Storage, ensuring that notes remain available even after the browser is refreshed. It also includes a dashboard and statistics page for better data visualization.



## Features

- Add new notes
- Edit existing notes
- Delete notes
- Search notes
- Organize notes by category
- Display date and time for each note
- Dark mode support
- Dashboard displaying:
  - Total Notes
  - Total Categories
  - Favorite Notes
- Statistics page with:
  - Pie Chart
  - Bar Chart
- Responsive user interface
- Local Storage support



## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Browser Local Storage

## Working

1. Open the application in a web browser.
2. Create a new note by entering the title, category, and description.
3. Save the note.
4. View all saved notes.
5. Edit or delete notes when required.
6. Search notes using keywords.
7. View dashboard statistics.
8. Analyze notes using the statistics page.



## Dashboard

The dashboard provides a summary of the stored notes, including:

- Total number of notes
- Total number of categories
- Total favorite notes



## Statistics

The statistics page uses Chart.js to visualize stored data.

- Pie Chart representing notes by category.
- Bar Chart representing the number of notes in each category.



## Data Storage

The application uses the browser's Local Storage to save notes locally. This allows users to access their notes even after refreshing or reopening the browser.
